Descripción de la práctica

En esta primera práctica se utiliza el estándar WebGL para desarrollar gráficos en el ordenador sin la necesidad de instalar un software adicional.

Para poder crear diferentes figuras en 3D, se ha utilizado la biblioteca three.js la cual permite simplificar el desarrollo de escenas WebGL.

Gracias a esta biblioteca, es posible crear escenas en una página web mediante el manejo de objetos gráficos tales como cámaras, luces, materiales o figuras geométricas.
