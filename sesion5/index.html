Descripción de la práctica

En esta practica se utilizará MPEG-DASH para la difusión de servicios de streaming de video. Para ello se utilizará un servidor web en una máquina virtual para ofrecer este servicio y para consumirlo se creará una página web que mediante la biblioteca Dash.js nos permitirá incrustar el video en streaming en la página.
